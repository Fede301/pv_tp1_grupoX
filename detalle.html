Detalles Del Proyecto

Objetivos Del Proyecto

Este proyecto tiene como objetivo principal el desarrollo de un entorno virtual de aprendizaje diseñado para centralizar el material de estudio y las actividades de las cátedras técnicas. Ante la dispersión de información en diferentes canales, se propone una estructura jerárquica que facilita el seguimiento pedagógico y la entrega de trabajos prácticos en tiempo y forma.

¿Como Lo Realizaremos?

La implementación se basara en metodologías ágiles para el desarrollo de software, priorizando la usabilidad y el acceso multiplataforma. Se utilizaron estándares de diseño responsivo para asegurar que el contenido académico sea accesible tanto en laboratorios de computación como en dispositivos móviles, integrando además un sistema de control de versiones para el trabajo colaborativo entre integrantes.
